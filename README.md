# 🏺 Drakotrypa Village Website

Μια σύγχρονη, responsive ιστοσελίδα αφιερωμένη στο ιστορικό χωριό **Δρακότρυπα Καρδίτσας**. Το project δημιουργήθηκε για να αναδείξει την πολιτιστική κληρονομιά, την ιστορία και τα τοπικά αξιοθέατα της περιοχής με μια premium αισθητική.

## ✨ Χαρακτηριστικά (Features)

* **Premium UI/UX:** Σχεδιασμός με έμφαση στην τυπογραφία (`Playfair Display` & `Inter`) και γήινη χρωματική παλέτα.
* **Interactive Timeline:** Ένα δυναμικό χρονολόγιο της ιστορίας του χωριού με custom JavaScript animations.
* **Curtain Transition Effects:** Ομαλές μεταβάσεις μεταξύ των σελίδων για μια "κινηματογραφική" εμπειρία χρήστη.
* **Split Screen Layout:** Ειδικά σχεδιασμένη σελίδα αξιοθεάτων με hover effects και δυναμικά overlays.
* **Fully Responsive:** Βελτιστοποιημένο για κινητά, tablets και desktops.
* **Weather API Integration:** Εμφάνιση καιρού σε πραγματικό χρόνο για την περιοχή της Δρακότρυπας μέσω του Open-Meteo API.
* **Interactive Components:** Accordions για την ιστορία, mobile-first navigation menu και scroll-reveal animations.

## 🛠️ Τεχνολογίες (Tech Stack)

* **HTML5:** Σημαντική δομή και προσβασιμότητα.
* **CSS3:** * Custom CSS Variables για εύκολη διαχείριση θεμάτων.
    * Flexbox & Grid Layouts.
    * Keyframe Animations & Glassmorphism εφέ.
* **JavaScript (Vanilla):** * Intersection Observer API για animations κατά το σκρολάρισμα.
    * Fetch API για την ανάκτηση δεδομένων καιρού.
    * Custom SPA-like navigation logic.

## 📂 Δομή Αρχείων

* `index.html`: Η κεντρική σελίδα (Hero, Καταστήματα, Περίληψη Ιστορίας).
* `history.html`: Αναλυτική ιστορική αναδρομή με το διαδραστικό χρονολόγιο.
* `sights.html`: Παρουσίαση αξιοθεάτων με split-screen design.
